Index: A Collection of Works Process Page

Sketches

I drew some sketches with reference to some personal homepages on the Internet and some initial ideas. I want my homepage to look simple and comfortable. I plan to add horizontally scrolling pictures of each item in a container, and jumping to different items by clicking on it. Exercises/projects/readings can be switched through the top buttons.

Title & Icons

Firstly, I wrote the title and my name, then added an overlay and three buttons on top. I found the website "fontawesome.com" very useful when working with icons for buttons. I just wrote "fa-solid fa-list" and the button looks like this "list" icon. After that I added some animation to the buttons, including hover, click, active-to increase the interactivity.

Container

Then I added a box to display all my assignments for this semester. I added three containers and made them show/hide when each button is clicked. At first I made it scroll horizontally, but eventually considering that horizontal scrolling took more time and was inconvenient, I changed it back to vertical scrolling and added an auto-scroll.

I took a screenshot of each assignment and put it into the container, and then added a title and link to the assignment to each picture.

Others

Finally, I added a background to the index. I used array and random functions to display a random background every time the page was opened. And added a horizontal scrolling animation to the background image. Then I added background sounds and a music player, and added different click sounds to the buttons, all from "pixabay.com".
